# Documentation: Chapter.cs

## Overview
`Chapter.cs` represents a logical grouping of lessons and quizzes within a course. it helps organize course content into a structured syllabus.

## Detailed Explanation
Each chapter belongs to a single course and contains an ordered list of lessons. it uses the `Order` property to ensure the syllabus is displayed in the correct sequence on the frontend.

## Program Flow
1. **Creation**: An admin adds a chapter to a course via the `CoursesController`.
2. **Organization**: `Lessons` and `Quizzes` are assigned to the chapter.
3. **Rendering**: The `CoursePlayerPage` on the frontend fetches chapters and their lessons to build the navigation sidebar.

## Connections
- **Connected to `Course`**: Child of a Course entity.
- **Connected to `Lesson`**: Parent of multiple Lesson entities.
- **Connected to `Quiz`**: Can contain quizzes for assessment.
- **Used by `CourseStructureRepository`**: To build the hierarchical view of the course.

## Properties Explanation
- `Id`: Primary key.
- `CourseId`: Foreign key linking to the parent Course.
- `Title`: The display name of the chapter (e.g., "Introduction to React").
- `Order`: An integer used to sort chapters (Chapter 1, Chapter 2, etc.).
- `Lessons` / `Quizzes`: Navigation collections to access sub-content.

## Why it is used
It provides a mid-level hierarchy for course content. Without chapters, a course with 50 lessons would have a messy, flat list that is hard for students to navigate.

## Interview Questions
1. **What is the role of the `[ForeignKey("CourseId")]` attribute?**
   *Answer: It explicitly tells Entity Framework that the `CourseId` property is the foreign key for the `Course` navigation property.*
2. **Why use an `Order` property instead of just sorting by `Id`?**
   *Answer: IDs are generated by the database and might not reflect the intended teaching order. An `Order` property allows instructors to rearrange chapters easily.*
3. **How would you fetch all chapters for a course along with their lessons in a single query?**
   *Answer: By using "Eager Loading" with the `.Include(c => c.Lessons)` method in EF Core.*
