# LearnSphere: Premium Learning Management System (LMS)

LearnSphere is a full-stack, state-of-the-art educational platform designed for seamless course delivery, real-time student tracking, and advanced academic assessments.

## ✨ Key Features
- **Sophisticated Player**: Supports Video, Audio, Image, and PDF content with automatic progress tracking.
- **Advanced Assessments**: A high-stakes testing engine with timed attempts, eligibility gates, and instant grading.
- **Formative Quizzes**: Chapter-end knowledge checks to reinforce learning.
- **Admin Suite**: Comprehensive tools for course creation, student management, and platform analytics.
- **Live Sessions**: Peer-to-peer real-time learning integration.

## 🛠️ Technology Stack
- **Backend**: C#, ASP.NET Core 8, Entity Framework Core (SQL Server).
- **Frontend**: JavaScript, React.js, Vite, Tailwind CSS, Lucide Icons.
- **Security**: JWT (JSON Web Tokens) for stateless authentication.
